# ☁️ Deploy RoofRoute to the Cloud (5 Minutes)

Want to access your app from anywhere without installing anything? Deploy it to the cloud for FREE!

## Option 1: Vercel (Easiest - Recommended)

### Step 1: Create Vercel Account
1. Go to https://vercel.com/signup
2. Click "Continue with GitHub" (or create account)
3. Verify your email

### Step 2: Deploy
1. Go to https://vercel.com/new
2. Click "Import Git Repository"
3. Paste your GitHub repo URL
4. Click "Import"
5. Click "Deploy"
6. **Done!** You'll get a live URL like `https://your-app.vercel.app`

---

## Option 2: Railway.app (Super Easy)

### Step 1: Create Account
1. Go to https://railway.app
2. Click "Start Project"
3. Connect your GitHub account

### Step 2: Deploy
1. Click "Deploy from GitHub"
2. Select your roofroute repository
3. Click "Deploy"
4. **Done!** Gets a live URL in seconds

---

## Option 3: Render.com (Free)

### Step 1: Create Account
1. Go to https://render.com
2. Sign up with GitHub

### Step 2: Deploy
1. Click "New +"
2. Select "Web Service"
3. Connect your GitHub repo
4. Set **Build Command:** `npm install`
5. Set **Start Command:** `npm start`
6. Click "Create Web Service"
7. **Done!** Live in 2-3 minutes

---

## If You Don't Have GitHub...

**Quick GitHub Setup (2 minutes):**

## Troubleshooting

### "Search doesn't work"
- Clear browser cache (Ctrl+Shift+Delete)
- Refresh page (Ctrl+R)
- Wait 2 minutes for deployment to finish

### "Can't deploy"
- Make sure all files are uploaded to GitHub
- Check that `server.js` and `package.json` are there
- Try a different hosting service

### "Want to use custom domain"
- Each service has custom domain options in settings
- Usually need to buy domain from GoDaddy, Namecheap, etc.
